# Java Array Matrix

Java-Übungsaufgaben zu **mehrdimensionalen Arrays und Matrizen** (Aufgaben 6–10).

---

## 📋 Aufgaben Übersicht

| Aufgabe | Thema | Beschreibung |
|---------|-------|-------------|
| 6 | 3×3 Matrix | Matrix mit Werten 1–9 erstellen und ausgeben |
| 7 | Zeilensummen | Summe jeder Zeile berechnen und ausgeben |
| 8 | Benutzereingabe | 2×2-Matrix einlesen, Original + Transponierte ausgeben |
| 9 | Hauptdiagonale | 4×4-Matrix mit Zufallswerten, nur Diagonale ausgeben |
| 10 | Zufallsmatrix | 5×5-Matrix, Gesamtsumme und größten Wert ermitteln |

---

## 💡 Verwendete Java-Konzepte

| Konzept | Verwendung |
|---------|-----------|
| `int[][]` | Zweidimensionale Arrays (Matrizen) |
| Verschachtelte `for`-Schleifen | Zeilen und Spalten durchlaufen |
| `Scanner` | Benutzereingabe (Aufgabe 8) |
| `Math.random()` | Zufallszahlen generieren (Aufgaben 9, 10) |
| Transponierte Matrix | Zeilen und Spalten tauschen `matrix[y][x]` |
| Bedingung `x == y` | Hauptdiagonale identifizieren |

---

## ▶️ Ausführen

```bash
javac AMatrixArray.java
java AMatrixArray
```

> ⚠️ Aufgabe 8 erwartet Benutzereingabe: 4 Zahlen für die 2×2-Matrix.
